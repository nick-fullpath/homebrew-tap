# homebrew-tap

Homebrew tap for [cursor-history](https://github.com/nick-fullpath/cursor-history) and other tools.

## Install

```bash
brew tap nick-fullpath/tap
brew install cursor-history
```

## Formulae

| Formula | Description |
|---|---|
| [cursor-history](https://github.com/nick-fullpath/cursor-history) | Browse, search, and resume Cursor Agent CLI sessions |
